# Windows Patch KB Retriever
This tool accepts a valid Windows KB (e.g. KB5009557) and returns the Windows KB that it is replacing. 

# Usage
```
$ win_patch_retriever/main.py KB5009557
Current: KB5009557
Replaces: KB5010196
```
